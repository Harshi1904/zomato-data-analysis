# Zomato Restaurant Data Analysis

## Overview
This project analyzes the Zomato restaurant dataset using Python and data visualization techniques.

## Tools Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- WordCloud
- Jupyter Notebook

## Analysis Performed
- Data Cleaning
- Missing Value Handling
- Restaurant Location Analysis
- Cuisine Analysis
- Rating Distribution
- Price vs Rating Analysis
- Correlation Heatmap
- Word Cloud Visualization

## Key Findings
- Popular locations contain the highest number of restaurants.
- North Indian and Chinese cuisines are highly popular.
- Most ratings fall between 3.5 and 4.5.
- Online ordering is widely available.
- Price and rating have a weak relationship.

## Recommendations
1. Focus marketing efforts on popular restaurant locations.
2. Promote highly rated restaurants with fewer votes.
3. Improve cuisine-based recommendations.
4. Encourage online ordering adoption.
5. Use ratings and votes for restaurant ranking.

## Conclusion
This project demonstrates how data analytics can provide business insights and improve decision-making.
